Introduction

Welcome to the first page of my critical reflective journal, written by Connor Orton as part my Indie Games Development master’s degree. My journal will document the progress of my practical work, contextual research and discuss my games development practices as I progress throughout my course. I will document the weekly challenges that I have been given in a critical way, explain how I think that I’ve accomplished the task and evaluate what I’ve learned in the process. This blog is designed for educational purposes, and any outsource material will be referenced correctly to the original source, author or creator at the bottom of each post. Below is a diagram that I created of the blog structure which highlights each module and how each post will be laid out in an agile project methodology format as this is how the course is managed. Below the diagram is a contents page that will take you to the specified course module.
