Pixel Pinball Arcade (2020)

I developed this game in a team of two which included myself and a 2D artist. It was written in the C# programming language. This was my second project working in a team of two.

Battle Beasts - RPG Mechanics Asset Pack (2017)

Battle Beasts RPG Mechanics asset pack is a collection of RPG mechanics that I created using Unreal Engine 4 with blueprint visual scripting. It was created as part as my programming module on my BSC Games Production undergraduate degree. The game mechanics included are inspired by the Pokemon game franchise. The main mechanics and systems the asset pack includes are:

- Combat system (Control your beasts against your opponent, use abilities and items to defeat them, features battles against npcs and random encounters.)
- Dialogue system (Features multi choice questions and multiple outcomes depending on dialogue choice.)
- In-Game database system (For showing the player what beasts exist in the game and if they have captured them)
- Engine System databases (To manage units, unit abilities, items etc.)
- Shop system (Combined with the dialogue system to allow the player to purchase items using currency from a merchant.)
- Bank system (Combined with the dialogue system to store player beasts in a unit storage)
- Trade system (Combined with the dialogue system to trade player beasts with NPCs.)
- Utility menu (To allow the player to access in-game database, beast team, inventory and options for game preferences.)
- Player unit progression (Beasts level up and learn new abilities depending on their level.)

Survival Training VR Simulation (2017)

Survival Training VR is a virtual reality simulator that I created in Unreal Engine 4 with Unreal Blueprint. This project was submitted alongside my dissertation for my BSC games production course. It featured a first person scenario where the player must learn survival skills in a virtual reality environment and answer questions in a quiz. At the end of the simulation the player was informed how they did on the quiz to let them know if they learned from the simulation.

Isolation Complex - Short Puzzle Game (2017)

Isolation complex is a short first person puzzle game that I created in Unreal Engine 4 using Unreal Blueprint. It was created to be submitted alongside my dissertation before I changed my idea to survival training in VR. It features puzzle solving and an inventory system which you can use to combine items.

Tower Defense Game Demo (2016)

Once Upon a Castle was a tower defense game that I created in Unreal Engine 4 using Unreal Blueprint. It features real time player combat with enemy waves which damage player defenses, resource gathering, tower building which featured the base and the weapon selections and more. Was showcased to students and some industry proffesionals on a showcase day at the end of my first year on my BSC course.
